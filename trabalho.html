Arquitetura de Software

O que é Arquitetura de Software?

É a estrutura do sistema que é composta por elementos de Software tanto externamente visiveis quanto das relações entre eles.

O que os profissionais da área fazem?

arquitetos de software mapeiam e definem estruturas dos sistemas a serem desenvolvidas,identificam problemas em potencial e apresentam soluções podendo atuar como lideres em quipes de TI.

Como está a área no mercado de trabalho?

Com a modernização dos negocios e sua migração para o ambiente digital a arquitetura de software vem se provando cada vez mais util e necessaria ampliando o mercado de trabalho da área conforme os softwares são criados a tendencia e o aparecimento de erros no entanot softwares criados sob a supervisão de arquitetos de software apresentam menos erros e se acontecem são mais faceis de serem modificados assim arquitetos de software trazem mais economia e produtividade para as empresas.

Onde os profissionais da área podem atuar?

arquitetos de software está entre as áreas mais disputadas no mercado de trabalho senão a mais disputada na área de informática podendo atuar como:

  • Desenvolvimento de softwares e codificação em linguagens de programação
  • IA's
  • Lei geral de proteção de dados
  • Analise de codigo fonte
  • Exploração de binarios
  • banco de dados relacional e não relacional
  • Google cloud e outros clouds
  • Identificação e solução de vulnerabilidades

Historia da Arquitetura de software

  • Em 1950 surgem os primeiros sistemas de software monoliticos com pouca modularidade.
  • Em 1970 surge a programação estruturada que trouxe abordagem mais organizada e controlada para o desenvolvimentos de software.
  • Em 1980 surge o paradigama orientado a objetos que revoluciona os sistemas desenvolvidos.
  • Em 1990 com o crescimento da intrnet surge a arquitetura cliente-servidor reponsavel pela interface do usuario a um servidor.
  • Atualmente está surgindo novas tendencias a área como as IA's e arquitetura em nuvem.
